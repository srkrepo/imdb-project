## Dataset from IMDB ##

### Prerequisite Steps ###

This section used to download and extract the IMDB dataset.
Once downloaded, execute unzip_tsv.gz.py file 

#### Extract and rename the file using pandas, gzip ####
1. Download dataset(tsv.gz) files from https://datasets.imdbws.com
`name.basics.tsv.gz`
`title.akas.tsv.gz`
`title.basics.tsv.gz`
`title.crew.tsv.gz`
`title.episode.tsv.gz`
`title.principals.tsv.gz`
`title.ratings.tsv.gz`
    
2. Open a terminal and run `python unzip_tsv_gz.py`. Wait until the following output shows up on the terminal.
    `Files unzipped/renamed successfully`
